# Images Folder

Place your images here!

## Required Images

### 1. Couple Photo (Required)
- **Filename**: `couple.jpg` (or update path in config.js)
- **Recommended Size**: 500x500px or larger (square aspect ratio works best)
- **Format**: JPG, PNG, or WebP
- **Description**: This is your main couple photo that appears for each day

### 2. Gallery Photos (Optional)
- **Filenames**: `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, etc.
- **Recommended Size**: 800x800px or similar
- **Format**: JPG, PNG, or WebP
- **Description**: These appear in the photo gallery

## Tips

- Keep file sizes under 2MB for faster loading
- Use descriptive names if you want
- Square images work best for the gallery
- The couple photo can be portrait, landscape, or square
- Compress images before uploading for best performance

## Image Compression Tools

- **Online**: TinyPNG, Squoosh, Compressor.io
- **Desktop**: ImageOptim (Mac), FileOptimizer (Windows)
